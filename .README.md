# Next-Gen Learning Dashboard

A futuristic, highly animated education platform featuring a bento-grid layout and real-time data fetching.

## Features
- **Bento Grid Layout**: A modern, responsive design using Tailwind CSS.
- **Data Fetching**: Utilizes Next.js Server Components (RSC) to fetch live data securely from Supabase.
- **Animations**: Implements Framer Motion for staggered page loads, hover states, and smooth spring physics.
- **Responsive Design**: Adapts seamlessly from desktop to mobile.

## Architectural Choices
- **Server/Client Split**: Used Server Components for data fetching to optimize performance and security. Used Client Components for Framer Motion interactions to handle user-specific animations.
- **Security**: Environment variables are handled securely via Vercel, with a `.env.example` template provided for configuration.

## Challenges
- Managing complex animations with Framer Motion while maintaining zero layout shifts.
- Configuring environment variables correctly for secure Supabase integration.
